Egy könyv kezelése - getOneBookControllersBackend.mjs

Ebben a részben szerkesztjük a controllers/allbooks mappában a getOneBookControllersBackend.mjs állományt, amelyben kidolgozzuk a CRUD Read művelethez tartozó logikát. Ez megfelel a http GET metódus-hoz tartozó logikának.

  • Szerkesszük a getOneBookControllersBackend.mjs állományunkat.
    1. import Book from '../../models/BookModelOnePicture.mjs'; - a Book modell alapértelmezett beimportálása.
    2. export const getOneBookBackend = async (req, res) => {...}; - hozzuk létre és vigyük ki nevesítve a http GET metódus-hoz tartozó logikát (Read a CRUD műveletekből).
    3. try {... res.statusCode = 200; return res.render('one_book/one_book.ejs', { book });} - a sikeres ág a 200-as http hibakód-dal. A book-t átadjuk a one_book/one_book.ejs állománynak.
    4. const { id } = req.params; - szedjük ki az id tulajdonságot a req.params-ból.
    5. const book = await Book.findById({ _id: id }); - keressük meg a könyvet és töröljük az adatbázisból.
    6. catch (error) { res.statusCode = 404; return res.render('404.ejs'); } - a sikertelen ág a 404-es http hibakód-dal.
