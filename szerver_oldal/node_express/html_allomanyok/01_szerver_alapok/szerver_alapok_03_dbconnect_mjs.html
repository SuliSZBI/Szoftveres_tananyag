Szerver alapok - dbConnect.mjs

Ebben a részben létrehozzuk az src mappában a dbConnect.mjs állományt, amelyben összekötjük a webszervert a Konyvtar adatbázissal.

  • Hozzuk létre az src mappában a dbConnect.mjs állományt. , és
    1. import app, { PORT } from './app.mjs'; - az app objektum alapértelmezett és a PORT nevesített beimportálása.
    2. import dbConnect from '../utils/dbConnection.mjs'; - a csatlakozási függvény beimportálása.
    3. dbConnect() - a csatlakozási promise meghívása.
    4. .then(() => {...}); - a fulfilled, azaz sikeres ág.
    5. app.listen(PORT, () => { console.log(`http://localhost:${PORT}/api`); }); - a szerver figyelőmódba állítása is itt történik meg.
    6. .catch((error) => { console.error(`A hiba oka: ${error.message}`); }); - a rejected, azaz sikertelen ág.
