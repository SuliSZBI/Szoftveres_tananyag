Fizetés beállítása - stripeRoutes.mjs

Ebben a részben szerkesztjük a routes mappában a stripeRoutes.mjs állományt, amelyben összesítjük a CRUD műveletekhez tartozó logikákat.

  • Szerkesszük a stripeRoutes.mjs állományunkat.
    1. import express from 'express'; - az express npm csomag beimportálása.
    2. import { stripePayment } from '../controllers/payment/stripeControllers.mjs'; - a stripePayment függvény nevesített importja.
    3. const router = express.Router(); - mivel a webszerver jelölésére az app csak a server.mjs állományban használható, ezért itt már csak a router szóval hivatkozhatunk rá, amit az express objektum Router() függvényével tehetünk meg.
    4. router.post('/create-checkout-session', stripePayment); - a http POST metódus-hoz tartozó logika hívása a stripePayment függvény segítségével.
    5. export default router; - a szerver hívás alapértelmezett kivitele.
