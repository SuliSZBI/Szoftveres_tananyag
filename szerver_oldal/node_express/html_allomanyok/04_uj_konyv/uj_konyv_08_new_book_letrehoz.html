Új könyv felvétele - new_book_letrehoz.mjs

Ebben a részben szerkesztjük az public/js mappában az új könyv felvételéhez szükséges new_book_letrehoz.mjs állományt.

  • Szerkesszük a new_book_letrehoz.mjs állományt.
    1. async function letrehoz(event) {...} - az új könyv létrehozásáért felelős aszinkron függvény.
    2. event.preventDefault(); - az adatok elküldése után ne frissüljön az oldal.
    3. const response = await fetch('/api/new-book', {...}); - a fetch hívása után kapott promise.
    4. method: 'POST', - adatokat küldünk a szervernek, tehát a http POST metódus-t használjuk.
    5. headers: { 'Content-Type': 'application/json' }, - annak jelölése, hogy a request body tartalmát JSON formátumban fogjuk küldeni.
    6. body: JSON.stringify({...}), - csomagoljuk is be.
    7. const resp = await response.json(); - a szervertől jövő válasz kicsomagolása, ha létezik.
    8. if (response.ok) { window.alert(resp.msg); window.location.href = '/api/books-backend'; } - ha sikeres a feltöltés, akkor ezt írjuk ki és menjünk át a /api/books-backend route-ra. Még nem létezik!
    9. else { window.alert(resp.msg); } - ha nem sikeres a feltöltés, akkor ezt írjuk ki.
  • Az eddigi munka letölthető formátumban: backend_03.rar
    Letöltés és kicsomagolás után a GitBash felületen a backend mappában adjuk ki a következő utasítást:
    npm install
  • A webszerver futtatásához adjuk ki a következő utasítást a GitBash felületen:
    node --watch ./src/server.mjs
