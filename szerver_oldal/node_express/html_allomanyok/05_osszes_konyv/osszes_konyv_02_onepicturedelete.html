Összes könyv kezelése - onePictureDeleteFromCloudinary.mjs

Ebben a részben szerkesztjük a middlewares mappában a onePictureDeleteFromCloudinary.mjs állományt, amelyben összekötjük a webszerverünket a Cloudinary felülettel és egy képet törölünk onnan.

  • Szerkesszük a onePictureDeleteFromCloudinary.mjs állományunkat.
    1. import dotenv from 'dotenv'; - a dotenv npm csomag beimportálása.
    2. dotenv.config(); - az értékeket a .env állományban keresse a backend mappában.
    3. import { v2 as cloudinary } from 'cloudinary'; - a cloudinary npm csomag beimportálása, mint v2.
    4. import Book from '../models/BookModelOnePicture.mjs'; - a Book modell alapértelmezett beimportálása.
    5. cloudinary.config({...}) - a Cloudinary oldalról vett értékek beállítása a .env állományból.
    6. const pictureDeleter = async (req, res, next) => {...}; - a pictureDeleter függvény létrehozása, amellyel törölhetjük a képet a felhőből.
    7. const { id } = req.params; - szedjük ki az id tulajdonságot a req.params-ból.
    8. const torolKep = await Book.findById({ _id: id }); - keressük meg a könyvet, amelyhez tartozó képet törölni akarjuk.
    9. const kep = torolKep.kep.split('/')[7].split('.')[0]; - szedjük ki a kép azonosítóját.
    10. await cloudinary.uploader.destroy(kep.toString()); - töröljük a képet a Cloudinary felületéről.
    11. next(); - lépjünk a következő middleware-re.
    12. export default pictureDeleter; - a pictureDeleter függvény alapértelmezett kivitele.
