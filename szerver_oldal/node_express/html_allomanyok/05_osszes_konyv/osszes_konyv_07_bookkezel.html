Összes könyv kezelése - book_kezel.mjs

Ebben a részben szerkesztjük az public/js mappában egy könyv törléséhez szükséges book_kezel.mjs állományt.

  • Szerkesszük a book_kezel.mjs állományt.
    1. async function torles(id) {...} - egy könyv törléséért felelős aszinkron függvény.
    2. const response = await fetch(`/api/books-backend/${id}`, {...}); - a fetch hívása után kapott promise.
    3. method: 'DELETE', - adatokat küldünk a szervernek, tehát a http DELETE metódus-t használjuk.
    4. headers: { 'Content-Type': 'application/json' }, - annak jelölése, hogy a request body tartalmát JSON formátumban fogjuk küldeni.
    5. const resp = await response.json(); - a szervertől jövő válasz kicsomagolása, ha létezik.
    6. if (response.ok) { window.alert(resp.msg); window.location.href = '/api/books-backend'; } - ha sikeres a törlés, akkor ezt írjuk ki és menjünk át a /api/books-backend route-ra.
    7. else { window.alert(resp.msg); } - ha nem sikeres a feltöltés, akkor ezt írjuk ki.
  • Az eddigi munka letölthető formátumban: backend_04.rar
    Letöltés és kicsomagolás után a GitBash felületen a backend mappában adjuk ki a következő utasítást:
    npm install
  • A webszerver futtatásához adjuk ki a következő utasítást a GitBash felületen:
    node --watch ./src/server.mjs
