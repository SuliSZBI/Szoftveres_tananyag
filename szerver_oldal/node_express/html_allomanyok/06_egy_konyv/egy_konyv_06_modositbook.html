Egy könyv kezelése - modosit_Book.ejs

Ebben a részben szerkesztjük a views/modosit_book mappában az összes könyv megjelenítéséhez szükséges ejs állományokat.

  • Szerkesszük a modosit_book.ejs állományt. és
    1. <link rel="stylesheet" href="/css/new_book.css" /> - mivel a server.mjs állományban beállítottuk a hivatkozást a public mappára, ezért minden elemére ilyen speciálisan tudunk hivatkozni. Most a new_book.css-re!!!
    2. <%- include('../nav.ejs') %> - beimportálása a nav.ejs állománynak.
    3. <%- include('tr_text.ejs', { td: 'Cím', id: 'cim', value: book.cim }) %> - beimportálása a tr_text.ejs állománynak, ahol átadjuk a { td: 'Cím', id: 'cim', value: book.cim } objektumot.
    4. <%- include('tr_number.ejs', { td: 'Oldalszám', id: 'oldalszam', value: book.oldalszam }) %> - beimportálása a tr_number.ejs állománynak, ahol átadjuk a { td: 'Oldalszám', id: 'oldalszam', value: book.oldalszam } objektumot.
    5. <%- include('tr_textarea.ejs', { td: 'Tartalom', id: 'tartalom', value: book.tartalom }) %> - beimportálása a tr_textarea.ejs állománynak, ahol átadjuk a { td: 'Tartalom', id: 'tartalom', value: book.tartalom } objektumot.
    6. <%- include('zsaner.ejs', { zsaner: book.zsaner }) %> - beimportálása a zsaner.ejs állománynak, ahol átadjuk a { zsaner: book.zsaner } objektumot.
    7. <%- include('kedvezmeny.ejs', { kedvezmeny: book.kedvezmeny }) %> - beimportálása a kedvezmeny.ejs állománynak, ahol átadjuk a { kedvezmeny: book.kedvezmeny } objektumot.
    8. <%- include('tr_regikep.ejs', { td: 'Régi kép', id: 'regikep', value: book.kep }) %> - beimportálása a tr_regikep.ejs állománynak, ahol átadjuk a { td: 'Régi kép', id: 'regikep', value: book.kep } objektumot.
    9. <%- include('tr_ujkep.ejs', { td: 'Új kép', id: 'ujkep' }) %> - beimportálása a tr_ujkep.ejs állománynak, ahol átadjuk a { td: 'Új kép', id: 'ujkep' } objektumot.
    10. <button type="submit" onclick="modosit(event, '<%= book._id %>', '<%= book.kep %>')">Módosít</button> - adjunk egy onclick="modosit(...)" eseménykezelőt a gombhoz, amivel módosítani tudjuk a könyvet.
    11. <script src="/js/book_modosit.mjs"></script> - hivatkozás beállítása a /js/book_modosit.mjs állományra.
  • Szerkesszük a tr_text.ejs, tr_textarea.ejs és tr_number.ejs állományokat. , és
    1. <td><%= td %>:</td> - a modosit_book.ejs-től kapott objektumból kivesszük a td értéket.
  • Szerkesszük a zsaner.ejs és kedvezmeny.ejs állományokat. és
    1. zsaner - a modosit_book.ejs-től kapott objektumból kivett érték.
    2. zsanerek - a getModositOneBookControllersBackend.mjs-től kapott objektumból kivett érték.
    3. kedvezmeny - a modosit_book.ejs-től kapott objektumból kivett érték.
    4. kedvezmenyek - a getModositOneBookControllersBackend.mjs-től kapott objektumból kivett érték.
  • Szerkesszük a tr_regikep.ejs és tr_ujkep.ejs állományokat. és
    1. <td><%= td %>:</td> - a modosit_book.ejs-től kapott objektumból kivesszük a td értéket.
