Vásárlás kezelése - PurchaseModel.mjs

Ebben a részben létrehozzuk a Purchase adatbázis modelünket.

  • Szerkesszük a PurchaseModel.mjs állományunkat.
    1. import mongoose from 'mongoose'; - a mongoose npm csomag beimportálása.
    2. const purchaseSchema = new mongoose.Schema({...}, {...}); - a MongoDB adatbázishoz tartozó gyűjtemény (collection) felépítési sémája.
    3. { timestamps: true } - időbélyező létrehozáskor és utolsó módosításkor.
    4. const Purchase = mongoose.model('purchase', purchaseSchema); - sémából létrehozott modell.
    5. export default Purchase; - a modell alapértelmezett kivitele.
