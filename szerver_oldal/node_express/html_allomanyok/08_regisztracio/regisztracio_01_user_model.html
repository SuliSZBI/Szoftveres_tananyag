Regisztráció kezelése - UserModel.mjs

Ebben a részben létrehozzuk a User adatbázis modelünket.

  • Szerkesszük a BookModelOnePicture.mjs állományunkat.
    1. import mongoose from 'mongoose'; - a mongoose npm csomag beimportálása.
    2. const userSchema = new mongoose.Schema({...}, {...}); - a MongoDB adatbázishoz tartozó gyűjtemény (collection) felépítési sémája.
    3. { timestamps: true } - időbélyező létrehozáskor és utolsó módosításkor.
    4. const User = mongoose.model('user', userSchema); - sémából létrehozott modell.
    5. export default User; - a modell alapértelmezett kivitele.
