Alap route-olás - mainControllersBackend.mjs

Ebben a részben szerkesztjük a controllers mappában a mainControllersBackend.mjs állományt, amelyben kidolgozzuk a CRUD Read művelethez tartozó logikát. Ez megfelel a http GET metódus-hoz tartozó logikának.

  • Szerkesszük a mainControllersBackend.mjs állományt.
    1. export const getMainBackend = (req, res) => {...}; - hozzuk létre és vigyük ki nevesítve a http GET metódus-hoz tartozó logikát (Read a CRUD műveletekből).
    2. try { res.statusCode = 200; return res.render('index.ejs'); } - a sikeres ág a 200-as http hibakód-dal.
    3. catch (error) { res.statusCode = 404; return res.render('404.ejs'); } - a sikertelen ág a 404-es http hibakód-dal.
