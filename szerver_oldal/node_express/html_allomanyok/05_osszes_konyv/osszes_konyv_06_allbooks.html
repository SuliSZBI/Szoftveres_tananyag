Összes könyv kezelése - all_Books.ejs

Ebben a részben szerkesztjük az views/all_books mappában az összes könyv megjelenítéséhez szükséges ejs állományokat.

  • Szerkesszük a nav.ejs állományt. és
  • Szerkesszük az all_books.ejs állományt.
    1. <link rel="stylesheet" href="/css/all_books.css" /> - mivel a server.mjs állományban beállítottuk a hivatkozást a public mappára, ezért minden elemére ilyen speciálisan tudunk hivatkozni.
    2. <%- include('nav.ejs') %> - beimportálása a nav.ejs állománynak.
    3. <%- include('tr_elem.ejs', { td: 'Cím', targy: elem.cim }) %> - beimportálása a tr_elem.ejs állománynak, ahol átadjuk a { td: 'Cím', targy: elem.cim } objektumot.
    4. <%- include('tr_kep.ejs', { td: 'Kép', targy: elem }) %> - beimportálása a tr_kep.ejs állománynak, ahol átadjuk a { td: 'Kép', targy: elem } objektumot.
    5. <%- include('szerzok.ejs', { td: 'Szerzők', targy: elem }) %> - beimportálása a zsaner.ejs állománynak, ahol átadjuk a { td: 'Szerzők', targy: elem } objektumot.
    6. <a class="modosit" href="/api/books-backend/<%= elem._id %>">Lekérés</a> - a következő részben kidolgozandó elemenkénti lekérése a könyveknek.
    7. <button class="torol" onclick="torles('<%= elem._id%>')">Törlés</button> - helyezzünk el egy onclick="torles(...)" eseménykezelőt a gombra.
  • Szerkesszük a tr_elem.ejs és tr_kep.ejs. és
    1. <td><%= td %>:</td> - a all_books.ejs-től kapott objektumból kivesszük a td értéket.
    2. <img src="<%= targy.kep %>" /> - a all_books.ejs-től kapott objektumból kivesszük az targy.kep értéket.
  • Szerkesszük a szerzo.ejs állományt.
    1. targy.szerzok - a all_books.ejs-től kapott objektumból kivesszük a targy.szerzok értéket.
