Új könyv felvétele - new_book.ejs

Ebben a részben szerkesztjük az views/new_book mappában az új könyv felvételéhez szükséges ejs állományokat.

  • Szerkesszük a nav.ejs állományt. és
  • Szerkesszük a new_book.ejs állományt. és
    1. <link rel="stylesheet" href="/css/new_book.css" /> - mivel a server.mjs állományban beállítottuk a hivatkozást a public mappára, ezért minden elemére ilyen speciálisan tudunk hivatkozni.
    2. <%- include('../nav.ejs') %> - beimportálása a nav.ejs állománynak.
    3. <%- include('tr_text.ejs', { td: 'Cím', id: 'cim' }) %> - beimportálása a tr_text.ejs állománynak, ahol átadjuk a { td: 'Cím', id: 'cim' } objektumot.
    4. <%- include('tr_number.ejs', { td: 'Oldalszám', id: 'oldalszam' }) %> - beimportálása a tr_number.ejs állománynak, ahol átadjuk a { td: 'Oldalszám', id: 'oldalszam' } objektumot.
    5. <%- include('tr_textarea.ejs', { td: 'Tartalom', id: 'tartalom' }) %> - beimportálása a tr_textarea.ejs állománynak, ahol átadjuk a { td: 'Tartalom', id: 'tartalom' } objektumot.
    6. <%- include('zsaner.ejs') %> - beimportálása a zsaner.ejs állománynak.
    7. <%- include('kedvezmeny.ejs') %> - beimportálása a kedvezmeny.ejs állománynak.
    8. <button onclick="letrehoz(event)">Létrehoz</button> - helyezzünk el egy onclick="letrehoz(event)" eseménykezelőt a gombra.
  • Szerkesszük a tr_text.ejs, tr_number.ejs és tr_textarea.ejs állományokat. , és
    1. <td><%= td %>:</td> - a new_book.ejs-től kapott objektumból kivesszük a td értéket.
    2. <input type="text" id="<%= id %>" /> - a new_book.ejs-től kapott objektumból kivesszük az id értéket.
  • Szerkesszük a zsaner.ejs és kedvezmeny.ejs állományokat. és
    1. <%= zsanerek[i] %> - a getNewBookControllersBackend.mjs-től kapott objektumból kivesszük a zsanerek értéket.
    2. <input type="text" id="<%= id %>" /> - a getNewBookControllersBackend.mjs-től kapott objektumból kivesszük a kedvezmenyek értéket.
  • Szerkesszük az new_book.css állományt.
