Vásárlás kezelése - purchaseRoutes.mjs

Ebben a részben szerkesztjük a routes mappában a purchaseRoutes.mjs állományt, amelyben összesítjük a CRUD műveletekhez tartozó logikákat.

  • Szerkesszük a purchaseRoutes.mjs állományunkat.
    1. import express from 'express'; - az express npm csomag beimportálása.
    2. import { postNewPurchase } from '../controllers/purchases/postNewPurchaseControllersBackend.mjs'; - a postNewPurchase függvény nevesített importja.
    3. import { getAllPurchases } from '../controllers/purchases/getAllPurchasesControllersBackend.mjs'; - a getAllPurchases függvény nevesített importja.
    4. const router = express.Router(); - mivel a webszerver jelölésére az app csak a server.mjs állományban használható, ezért itt már csak a router szóval hivatkozhatunk rá, amit az express objektum Router() függvényével tehetünk meg.
    5. router.get('/', getAllPurchases); - a http GET metódus-hoz tartozó logika hívása a getAllPurchases függvény segítségével.
    6. router.post('/', postNewPurchase); - a http POST metódus-hoz tartozó logika hívása a postNewPurchase függvény segítségével.
    7. export default router; - a szerver hívás alapértelmezett kivitele.
