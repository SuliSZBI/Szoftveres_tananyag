Új könyv felvétele - BookModelOnePicture.mjs

Ebben a részben létrehozzuk a Book adatbázis modelünket.

  • Szerkesszük a BookModelOnePicture.mjs állományunkat.
    1. import mongoose from 'mongoose'; - a mongoose npm csomag beimportálása.
    2. const bookSchema = new mongoose.Schema({...}, {...}); - a MongoDB adatbázishoz tartozó gyűjtemény (collection) felépítési sémája.
    3. { timestamps: true } - időbélyező létrehozáskor és utolsó módosításkor.
    4. const Book = mongoose.model('book', bookSchema); - sémából létrehozott modell.
    5. export default Book; - a modell alapértelmezett kivitele.
