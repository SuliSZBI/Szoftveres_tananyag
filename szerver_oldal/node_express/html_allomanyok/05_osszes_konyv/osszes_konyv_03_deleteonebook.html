Összes könyv kezelése - deleteOneBookControllersBackend.mjs

Ebben a részben szerkesztjük a controllers/allbooks mappában a deleteOneBookControllersBackend.mjs állományt, amelyben kidolgozzuk a CRUD Delete művelethez tartozó logikát. Ez megfelel a http DELETE metódus-hoz tartozó logikának.

  • Szerkesszük a deleteOneBookControllersBackend.mjs állományunkat.
    1. import Book from '../../models/BookModelOnePicture.mjs'; - a Book modell alapértelmezett beimportálása.
    2. export const deleteOneBookBackend = async (req, res) => {...}; - hozzuk létre és vigyük ki nevesítve a http DELETE metódus-hoz tartozó logikát (Delete a CRUD műveletekből).
    3. try {... res.statusCode = 200; return res.json({ msg: 'Sikeres törlés!' });} - a sikeres ág a 200-as http hibakód-dal.
    4. const { id } = req.params; - szedjük ki az id tulajdonságot a req.params-ból.
    5. await Book.findByIdAndDelete({ _id: id }); - keressük meg a könyvet és töröljük az adatbázisból.
    6. catch (error) { res.statusCode = 500; return res.json({ msg: 'Valami hiba történt!' }); } - a sikertelen ág a 500-as http hibakód-dal.
