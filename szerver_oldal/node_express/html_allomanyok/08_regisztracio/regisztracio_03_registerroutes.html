Regisztráció kezelése - userRoutes.mjs

Ebben a részben szerkesztjük a routes mappában a userRoutes.mjs állományt, amelyben összesítjük a CRUD műveletekhez tartozó logikákat.

  • Szerkesszük a userRoutes.mjs állományunkat.
    1. import express from 'express'; - az express npm csomag beimportálása.
    2. import { registerCustomer } from '../controllers/users/registerControllers.mjs'; - a registerCustomer függvény nevesített importja.
    3. const router = express.Router(); - mivel a webszerver jelölésére az app csak a server.mjs állományban használható, ezért itt már csak a router szóval hivatkozhatunk rá, amit az express objektum Router() függvényével tehetünk meg.
    4. router.post('/register', registerCustomer); - a http POST metódus-hoz tartozó logika hívása a registerCustomer függvény segítségével.
    5. export default router; - a szerver hívás alapértelmezett kivitele.
