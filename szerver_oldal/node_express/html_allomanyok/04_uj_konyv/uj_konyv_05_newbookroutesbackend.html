Új könyv felvétele - newBookRoutesBackend.mjs

Ebben a részben szerkesztjük a routes mappában a newBookRoutesBackend.mjs állományt, amelyben összesítjük a CRUD műveletekhez tartozó logikákat.

  • Szerkesszük a newBookRoutesBackend.mjs állományunkat.
    1. import express from 'express'; - az express npm csomag beimportálása.
    2. import pictureUploader from '../middlewares/onePictureToCloudinary.mjs' - a pictureUploader függvény alapértelmezett importja.
    3. import { getNewBookBackend } from '../controllers/newBook/getNewBookControllersBackend.mjs'; - a getNewBookBackend függvény nevesített importja.
    4. import { postNewBookBackend } from '../controllers/newBook/postNewBookControllersBackend.mjs'; - a postNewBookBackend függvény nevesített importja.
    5. const router = express.Router(); - mivel a webszerver jelölésére az app csak a server.mjs állományban használható, ezért itt már csak a router szóval hivatkozhatunk rá, amit az express objektum Router() függvényével tehetünk meg.
    6. router.get('/', getNewBookBackend); - a http GET metódus-hoz tartozó logika hívása a getNewBookBackend függvény segítségével.
    7. router.post('/', pictureUploader, postNewBookBackend); - a http POST metódus-hoz tartozó logika hívása a pictureUploader middleware és a postNewBookBackend függvény segítségével.
    8. export default router; - a szerver hívás alapértelmezett kivitele.
