Új könyv felvétele - getNewBookControllersBackend.mjs

Ebben a részben szerkesztjük a controllers/newBook mappában a getNewBookControllersBackend.mjs állományt, amelyben kidolgozzuk a CRUD Read művelethez tartozó logikát. Ez megfelel a http GET metódus-hoz tartozó logikának.

  • Szerkesszük az adatok.mjs állományunkat a public/js mappában.
    1. export { zsanerek, kedvezmenyek }; - zsanerek és kedvezmenyek nevesített exportja.
  • Szerkesszük a getNewBookControllersBackend.mjs állományunkat.
    1. import { zsanerek, kedvezmenyek } from '../public/js/adatok.mjs'; - a zsánerek és kedvezmények nevesített beimportálása.
    2. export const getNewBookBackend = (req, res) => {...}; - hozzuk létre és vigyük ki nevesítve a http GET metódus-hoz tartozó logikát (Read a CRUD műveletekből).
    3. try { res.statusCode = 200; return res.render('new_book/new_book.ejs', { zsanerek, kedvezmenyek }); } - a sikeres ág a 200-as http hibakód-dal. A zsánereket és kedvezményeket átadjuk a new_book/new_book.ejs állománynak.
    4. catch (error) { res.statusCode = 404; return res.render('404.ejs'); } - a sikertelen ág a 404-es http hibakód-dal.
